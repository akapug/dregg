deos-editor surface

The deos code editor (deos-zed) — a rope-backed, syntax-highlighting editor over the Fs trait (FirmamentFs seam: a file is a cell, a save is a receipted turn). The buffer IS a document-language document: editing accrues verifiable patches, so the durable form is the patch history, not flat bytes.
