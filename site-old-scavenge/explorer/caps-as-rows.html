Your capabilities, expressed as the rows you may read.

A dregg capability is a database view. Present a token below and watch the cell rows narrow — the same Row-Level-Security gate the node enforces, made legible. A token confined to a resource prefix sees a strict subset; the no-amplification property, observed, not asserted. This is the browser twin of the pg-dregg cap-gated query cookbook (sql/cookbook.sql) — one glass in SQL, one here.
