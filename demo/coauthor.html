The Commons Forge

A crowd co-authors one shared dungeon draft — by vote. Anyone proposes a bounded, typed edit (add a room, an exit, an item, retarget the objective); the seated co-authors quorum-vote; and a certified edit is handed to the validator. A voted-for edit that would break the world — a dangling exit, an unreachable goal — is refused and rolled back even though it won the vote. The crowd proposes; the world resolves. The draft grows, and stays playable at every step.

What is actually real here

      
    
        
  • The crowd proposes a bounded, typed edit — a closed set (AddRoom / AddExit / PlaceItem / SetObjective), not free-form DSL. An edit is a typed value, not prose.
    • 
        
  • The quorum certifies the winning edit on the real collective-choice engine — each ballot a WriteOnce cap-bounded turn, the tally Monotonic, a round certifying only once the polis AffineLe quorum gate (M = 3 of 5) admits the decision-turn. A quorum-met close emits a verifiable quorum certificate, and an independent light-client replay recomputes the same board.
    • 
        
  • The validator disposes. The certified edit is re-validated through parse_dungeon (the same fail-closed gate the forge runs): a sound edit is applied (the draft grows + still plays), a breaking one is refused and rolled back — despite the passing vote. The crowd cannot vote a broken world into being.
    • 
        
  • The draft is playable at every step — the rendered .dungeon source parses into a real GameSession over the same /game path. The edit history is append-only, certified provenance.
    • 
      

      
Honest scope: bounded structured edits (not free-form DSL editing); the quorum-certified tally is over demo identities (each seat's key is blake3(name)); the draft is server memory. A production deployment binds each seat to a real custody key + a signed ballot and persists the draft. The quorum mechanism, the WriteOnce ballots, the Monotonic tally, the light-client recomputation, and the fail-closed validator are all real.
