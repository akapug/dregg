The scene language, in one glance

--- id / title ---
Frontmatter. Between two --- fences at the top: id: and title:. Every story needs one.
=== passage_name
A passage. A named section of the story. The first passage is where play begins; prose below it is the narration.
* [choice text]
A choice. One line per option. Taking it is a real verified turn.
* [text] { gate }
A gated choice. The condition (e.g. { lit >= 1 }) goes on the SAME line as the choice. Shown but not takeable until it holds.
~ var = value
An effect. Under a choice: ~ x = 1, ~ x += 2, ~ x -= 1. Sets story state the gates read.
-> target
Navigation. Where a choice leads. -> END (or prose ending a leaf passage) ends the story.
