=== loadedRel_def ===   (the @load_vec FFI postcondition, packaged and ASSUMED)
[oracles: DISK_THM] [axioms: ] []
⊢ ∀input bufAddr s.
    loadedRel input bufAddr s ⇔
    FLOOKUP s.locals «len» = SOME (ValWord (n2w (LENGTH input))) ∧
    FLOOKUP s.locals «base» = SOME (ValWord bufAddr) ∧
    memRel input bufAddr s ∧ LENGTH input < 2 ** 63 ∧
    EVERY (λx. x < 256) input
