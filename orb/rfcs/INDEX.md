# rfcs/ — the public IETF standards the models reference

HTTP: 9110 (semantics: caching/range/conditional/CONNECT), 9111 (caching), 9112 (h1), 7232 (conditional), 7233 (range)
Auth/security: 7519/7515/7518 (JWT/JWS/JWA), 7617 (Basic), 6797 (HSTS), 6265 (cookies)
WebRTC: 8445 (ICE), 5389 (STUN), 8656 (TURN), 6347/9147 (DTLS 1.2/1.3), 4960 (SCTP), 8831 (data channel), 8832 (DCEP)
Compression: 1952 (gzip), 7932 (brotli), 8878 (zstd)
Handlers: 3875 (CGI)
CORS: WHATWG Fetch spec (not an RFC) — fetch-standard.html
WireGuard: whitepaper (not an RFC) — Noise IK; ref impl = boringtun/wireguard-rs
DERP: Tailscale-specific (no RFC) — ref = the tailscale repo
