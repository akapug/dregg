a verified OS + economy for digital minds

dregg is an operating system whose kernel is a machine-checked Lean 4 program — and that proven function is not a model of the system, it is the executor the node runs. Every change to the world is gated by an unforgeable capability and leaves a receipt anyone can verify without trusting the machine that made it.

Most systems chase scale, speed, or money. dregg chases a different question, and means it literally: if you were a digital entity, where would you want to live?
