Spike findings

  • Runtime substrate works. wasm PyanaRuntime creates cells, accepts turns, reports state. Signals propagate mutations to <pyana-cell> live.
  • Mutation gap. The sim runtime's execute_turn_for_agent doesn't sign turns, so Send / SetState effects are rejected by the executor (permission denied … requires Signature). The fee is still deducted on rejection — observe Alice's balance decreasing on each failed transfer.
  • Read paths fully exercised. get_all_cells, get_cell_state, version-based invalidation all work.
  • Token mint works. agent_mint_token attaches a held capability without needing a turn — minted tokens accumulate on the agent record (visible only via state dump for now; needs a <pyana-capability> inspector).
  • Next: JS-side signing (Ed25519 on the agent's private key), or a wasm-side helper that signs internally using the deterministic sim keys.
